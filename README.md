# Roguelike ncurses
This is my try on a ASCII terminal roguelike in C with the help of ncurses. 
With loads of epic content such as being able to fight one enemy in the only room. 
## Build and Run

Clone the repo:
```bash
git clone https://github.com/Ventt1x/roguelike_ncurses.git
cd roguelike_ncurses
```
Compile:
```bash
make
```
Enjoy:
```bash
./roguelike
```

## Controls
**Player (you):**
- wasd to move
- arrow keys to shoot
- Ctrl+C to quit

**Enemies (not you):**

They move on their own you idiot
